# To run this application

##Console:

```python -m venv .venv```

```.venv\acripts\activate```

``pip install -r requirements.txt``

``cp contrib/env-sample .env``

`` python manage.py makemigrations``

```python manage.py migrate```

``python manage.py runserver``
